# v0.1.1

## Highlights
- Environment profiles via `.omaken/envs` with a TUI selector and default input values.
- Folder widgets load asynchronously with a loading indicator to keep navigation responsive.
- Search with background indexing plus schema previews.

## Added
- Environment profile loader with active selection stored in `.omaken/envs/active`.
- Env template file to help create new profiles.
- Search index (SQLite) and a search screen with results preview.
- Tags support in script schemas.

## Changed
- Azure scripts moved under `.omaken` and the Azure widget shows richer account metadata.
- TUI navigation hints updated (Alt+E for environments).
- Workspace config version set to v0.1.1.
- History artifacts and env configs are ignored by git (template kept).
- `.omaken/envs` is hidden from TUI navigation.

## Fixed
- UI no longer stalls while loading heavy Lua widgets.
